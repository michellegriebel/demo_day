jQuery is a JavaScript library

Why use jQuery?  jQuery is fast and lightweight which moves in the HTML DOM (Document Object Model) to make animations, manipulates page content, changes the style and provides cool user experience effects.

Differences  jQuery is a library built in the JavaScript language that helps to use the JavaScript language.  

Popularity  jQuery has become the most popular because it is so easy to use and extremely powerful.

Starting Point of Code  The starting point of jQuery code execution is $(document).ready() function which is executed when the DOM is loaded.
